Программа поиска Twitter пользователей (modules, layers, opt)
Исходные параметры - список twitter акканутов, чьих followers нужно просмотреть на предмет соответствия критериям. 
Список ключевых слов.
Критерии для проверки фолловера на соответствие - в его профиле и постах есть одно из списка ключевых слов.
Язык реализации - любой. Оптимальны - python 3, nodejs javascript, php
Подключение по API
У API есть лимит в число запросов каждые 15 минут.
Программа должна делать запросы в рамках лимита и засыпать на 15 минут.
Отличным уровнем реализации является сохранение состояния и продолжение с места прошлой остановки.
Целевым уровнем реализации является архитектура с очередью сообщений и свободным включением «воркеров», которые смогут делать лимит в 15 минут * число запущенных скриптов.
Требуется показать процесс работы программы, описать ее архитектуру, показать любой участок исходного кода.
